Detailed Description

A class that parses a Text file which will be use to post to WordPress.

Parses a text file nakes a post to WordPress. By passing the sanitation routines done by WordPress wp_insert_post function through a database update.

Remarks
Please make sure that "MY_WP_PATH" is properly defined to point to your WordPress "wp_load.php" location, This class relies heavily on the constants defined for database access in WordPress configuration file.
